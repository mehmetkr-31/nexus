## Requirements

- **Bun** ≥ 1.1 (or Node.js ≥ 20)
- **TypeScript** ≥ 5 with `strict: true`
- **React** ≥ 18 (19 recommended) — only for `@nexus-framework/react`
- Canton JSON Ledger API v2 endpoint

## Install

<Tabs items={["Core only", "Core + React"]}>
  <Tab value="Core only">
  ```bash
  bun add @nexus-framework/core
  ```
  </Tab>
  <Tab value="Core + React">
  ```bash
  bun add @nexus-framework/core @nexus-framework/react @tanstack/react-query
  ```
  </Tab>
</Tabs>

## Canton Sandbox (local development)

Start a Canton Sandbox with Docker:

```bash
docker run -p 7575:7575 digitalasset/canton-sandbox:latest
# JSON Ledger API → http://localhost:7575
# OpenAPI schema  → http://localhost:7575/docs/openapi
```

Sandbox accepts **HMAC256 tokens** via `sandboxAuth()` — no RS256 key management needed in development.

## TypeScript config

```json title="tsconfig.json"
{
  "compilerOptions": {
    "strict": true,
    "moduleResolution": "bundler",
    "module": "Preserve"
  }
}
```

## Next.js config

`transpilePackages` is required in `next.config.ts`:

```ts title="next.config.ts"
const nextConfig = {
  transpilePackages: ['@nexus-framework/core', '@nexus-framework/react'],
};
```
