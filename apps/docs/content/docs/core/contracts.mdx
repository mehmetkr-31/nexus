Contract queries are accessed via `nexus.ledger.contracts`.

## fetchActiveContracts

Fetches the first page of active contracts matching a template and optional filters.

```ts
const response = await nexus.ledger.contracts.fetchActiveContracts<IouPayload>({
  templateId: 'my-package:Iou:Iou',
  parties: ['Alice::122059a10c...'],   // optional — filter by party
  filter: { currency: 'USD' },         // optional — payload filter
});

// response.contracts: ActiveContract<IouPayload>[]
// response.nextPageToken?: string
```

### ActiveContract

```ts
interface ActiveContract<T = Record<string, unknown>> {
  contractId: string;
  templateId: TemplateId;   // { packageId, moduleName, entityName }
  payload: T;               // Daml-LF JSON-encoded contract payload
  createdAt: string;        // ISO 8601
  signatories: string[];
  observers: string[];
}
```

## fetchAllActiveContracts

Automatically follows pagination cursors to collect all contracts.
Use with care on large datasets.

```ts
const contracts = await nexus.ledger.contracts.fetchAllActiveContracts<IouPayload>({
  templateId: 'my-package:Iou:Iou',
});
// returns ActiveContract<IouPayload>[]
```

## fetchContractById

```ts
const contract = await nexus.ledger.contracts.fetchContractById<IouPayload>(
  'contractId-abc123',
);
// returns ActiveContract<IouPayload> | null
```

## Generic types

All methods accept a type parameter `T` for the contract payload:

```ts
interface IouPayload {
  owner: string;
  amount: string;
  currency: string;
}

const { contracts } = await nexus.ledger.contracts.fetchActiveContracts<IouPayload>({
  templateId: 'my-package:Iou:Iou',
});

// contracts[0].payload.amount is string ✓
```
