**Nexus Framework** is a modular SDK that bridges the Canton JSON Ledger API v2 with the modern React ecosystem. Inspired by [better-auth](https://better-auth.com), every capability is a **plugin** — auth strategies, query hooks, and real-time streaming are all opt-in and composable.

## Architecture

```
Your App (Next.js / React)
       │
       ▼
createNexusClient({ plugins: [sandboxAuth(), tanstackQueryPlugin(), streamingPlugin()] })
       │
       ▼
@nexus-framework/core   ← CantonClient, ContractQuery, CommandSubmitter
       │
       ▼
Canton JSON Ledger API v2
```

## Features

| Feature | Description |
|---------|-------------|
| **Plugin-based auth** | `sandboxAuth()`, `jwtAuth()`, `oidcAuth()` — swap without touching your app code |
| **Contract queries** | Typed ACS queries, pagination, `fetchContractByKey`, Daml interface views |
| **Choice return values** | `exerciseAndGetResult<TResult>()` — typed Daml return values |
| **TanStack Query** | `tanstackQueryPlugin()` adds `useContracts`, `useInterface`, `useExerciseAndGetResult` and more |
| **Real-time streaming** | `streamingPlugin()` adds `useStreamContracts` — live WebSocket ACS updates |
| **SSR prefetch** | `prefetchContracts` + `HydrationBoundary` for Next.js App Router |
| **Type-safe** | Zod-validated API responses, full TypeScript inference |
